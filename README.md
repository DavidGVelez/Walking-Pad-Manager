# Walking Pad Manager

App móvil creada con React Native y Expo para gestionar una walking pad.

## Requisitos

- Node.js 20 o superior
- npm
- App Expo Go en iOS o Android para probar en dispositivo físico

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm start
```

Después escanea el QR con Expo Go, o usa:

```bash
npm run ios
npm run android
npm run web
```
